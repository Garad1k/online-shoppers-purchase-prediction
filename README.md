# Анализ поведения пользователей интернет-магазина и прогнозирование вероятности покупки

## Описание проекта

Проект посвящен анализу поведения пользователей интернет-магазина и построению модели прогнозирования вероятности совершения покупки с использованием Python.

В качестве исходных данных используется набор Online Shoppers Purchasing Intention Dataset, содержащий сведения о пользовательских сессиях интернет-магазина.

## Цель проекта

Провести анализ поведения пользователей интернет-магазина и построить модель классификации, позволяющую прогнозировать вероятность совершения покупки на основе признаков пользовательской сессии.

## Используемые инструменты

- Python
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn
- Jupyter Notebook

## План реализации

1. Загрузка и первичный анализ данных.
2. Предварительная обработка данных.
3. Исследовательский анализ и визуализация.
4. Построение модели классификации.
5. Оценка качества модели.
6. Интерпретация результатов и выводы.
